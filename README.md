📡 О проекте
FRN Web Control — это удобный веб-интерфейс для управления вашим FRN-линком. Позволяет удалённо подключаться к серверам FRN, переключать комнаты, настраивать RX GPIO и отслеживать статус подключения.

✨ Возможности
🖥️ Веб-интерфейс — управление с любого устройства в сети
🔌 Поддержка GPIO — мониторинг RX сигнала (GPIO2) с настройкой длительности
🌍 14+ серверов FRN — предустановленные серверы с комнатами
🔄 Быстрое переключение — смена сервера и комнаты в один клик
📊 Статус подключения — визуальный индикатор состояния линка
🛠️ CLI-команды — управление через терминал
